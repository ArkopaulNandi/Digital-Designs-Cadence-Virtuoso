# Digital-Designs-Cadence-Virtuoso
This repository contains all my digital designs at CMOS Level, designed with the Cadence Virtuoso Tool.
The intial designs are done with the 180nm technology node.
The Designs include:
1. CMOS Inverter
2. CMOS Nand Gate
3. CMOS Nor Gate
4. CMOS Xnor Gate
5. CMOS Xor Gate
6. CMOS AND Gate
7. CMOS OR Gate
8. CMOS Full Adder Circuit
9. 4-bit CMOS Ripple Adder
